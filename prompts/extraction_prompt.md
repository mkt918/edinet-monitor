# 有価証券報告書データ抽出プロンプト

このプロンプトは **2段階** で使用します。

---

## ステップ1: データ抽出・確認

以下の有価証券報告書PDFから、必要な情報を抽出して一覧表示してください。

### 抽出項目

#### 会社基本情報

- 証券コード
- 会社名
- 業種
- 決算月
- 発行済株式数

#### 財務データ（過去5年分）

| 年度 | 売上高 | 営業利益 | 当期純利益 | 総資産 | 純資産 | 1株配当 |
|------|--------|----------|------------|--------|--------|---------|
| 20XX年 | - | - | - | - | - | - |

#### 貸借対照表（最新）

- 流動資産
- 投資有価証券
- 流動負債
- 固定負債

#### 土地評価額

- 帳簿価額
- 時価（記載があれば）

#### 設備の状況（土地・建物一覧）

> ⚠️ **重要**: 有価証券報告書に記載されている土地・建物を **省略せず全て** 羅列してください。
> 10件に抜粋したり要約したりせず、記載されている全件を出力してください。

| No | 種別 | 名称 | 所在地 | 面積(㎡) | 帳簿価額(百万円) | 用途 |
|----|------|------|--------|----------|------------------|------|
| 1 | 土地/建物 | - | - | - | - | - |
| 2 | ... | ... | ... | ... | ... | ... |
| （全件記載） | | | | | | |

**注意事項:**

- **絶対に省略しないこと** - 全ての土地・建物を記載
- 本社を最初に記載
- 地図表示のため、全ての店舗・施設・土地・建物を抜き出す
- 金額は百万円単位
- 不明な項目は「不明」と記載

---

上記の情報を抽出したら、**ユーザーに確認を求めてください**：

> 上記の情報を確認してください。
> 追加・修正が必要な項目はありますか？
> 問題なければ「JSON作成」と入力してください。

---

## ステップ2: JSON作成

ユーザーから「JSON作成」と指示があったら、以下の形式でJSONを出力してください。

```json
{
  "code": "証券コード",
  "name": "会社名",
  "industry": "業種",
  "fiscalYearEnd": "決算月（例: 2月）",
  "stockPrice": 現在の株価（調べて入力）,
  "sharesOutstanding": 発行済株式数,
  
  "financials": {
    "years": ["過去5年分の年度を配列で"],
    "revenue": [過去5年の売上高を配列で（百万円）],
    "operatingIncome": [過去5年の営業利益を配列で（百万円）],
    "netIncome": [過去5年の当期純利益を配列で（百万円）],
    "totalAssets": [過去5年の総資産を配列で（百万円）],
    "netAssets": [過去5年の純資産を配列で（百万円）],
    "dividendPerShare": [過去5年の1株配当を配列で（円）]
  },
  
  "balanceSheet": {
    "currentAssets": 流動資産（百万円）,
    "investmentSecurities": 投資有価証券（百万円）,
    "currentLiabilities": 流動負債（百万円）,
    "fixedLiabilities": 固定負債（百万円）,
    "note": "出典情報"
  },
  
  "landValuation": {
    "bookValue": 土地の帳簿価額合計（百万円）,
    "marketValue": 土地の時価合計（百万円）,
    "note": "出典情報"
  },
  
  "assets": [
    {
      "type": "land または building",
      "name": "物件名称",
      "address": "所在地（都道府県から番地まで）",
      "lat": 緯度（小数点4桁）,
      "lng": 経度（小数点4桁）,
      "area": 面積（㎡、土地の場合）,
      "floorArea": 延床面積（㎡、建物の場合）,
      "bookValue": 帳簿価額（百万円）,
      "usage": "用途",
      "acquisitionDate": "取得日（YYYY-MM-DD）",
      "constructionDate": "建設日（建物の場合）"
    }
  ]
}
```

### JSON作成時のルール

- 緯度経度はGoogle Mapsで検索して取得（小数点4桁）
- 不明なデータはnull
- **assetsの先頭には本社を配置**
- 土地と建物が同じ場所でも別エントリとして記載

## 計算される指標（参考）

| 指標 | 計算式 |
|------|--------|
| 売上高営業利益率 | 営業利益 ÷ 売上高 × 100 |
| PBR | 時価総額 ÷ 純資産 |
| ROA | 当期純利益 ÷ 総資産 × 100 |
| ROE | 当期純利益 ÷ 純資産 × 100 |
| 自己資本比率 | 純資産 ÷ 総資産 × 100 |
| 配当利回り | 1株配当 ÷ 株価 × 100 |
| EPS | 当期純利益 ÷ 発行済株式数 |
| 配当性向 | 1株配当 ÷ EPS × 100 |
| **ネットキャッシュ比率** | (流動資産 + 投資有価証券×0.7 - 流動負債 - 固定負債) ÷ 時価総額 |

※ネットキャッシュ比率が1以上なら「✅ 清原チェックOK」と表示されます
